# Simple To-Do List (HTML/CSS/JS)

Open `index.html` in your browser to use the app.

- Add tasks using the input + `Add` button (or press `Enter`)
- Set an optional due date for each task
- Choose a priority: High, Medium, or Low
- Mark tasks complete using the checkbox
- Delete tasks using the `Delete` button next to each item

This app uses only plain HTML, CSS, and JavaScript (no frameworks).
